== Description ==
 Integrates Areaexpress shipping  with your website to handle your order delivery.  Calculate shipping rates. Change order status to ship via Areaexpress to send delivery request to Areaexpress. It's seamless delivery, on demand.

Note - Once you change order status to ship via areaxpress, order data is sent to areaexpress.ng
 

== Installation ==
1. Upload folder `areaexpress.zip` to the `/wp-content/plugins/` directory
2.Unzip folder
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Ready. Now add  Api key and store name to your site through the Areaexpress menu by the side menu
5.Set your Woocommerce store location  to a location in Nigeria. e.g Nigeria-Lagos, city: lekki,ikota
6.You are set to go. NB:: Api key can be found on your areaexpress profile


== Frequently Asked Questions ==
How do i send delivery request?

Change your order status to Ship via Areaexpress.

How to  i get my Api key?

On your areaexpress  dashboard
